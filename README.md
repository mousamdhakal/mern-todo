# mern-todo

Todo list with authentication and login using MERN stack

[Live site](https://mern-todo.netlify.app/)

Tech and libraries used:

### Frontend (client)
---
- React 
- Redux
- Axios

### Backend (server)
--- 
- Nodejs
- Express
- Mongodb(database) 
- Mongoose
- JWT for authentication

### Hosting
---
- React App -> Netlify
- Node server -> Heroku
- Mongodb Database -> Mongo Atlas
